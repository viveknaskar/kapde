Kapde is a modern and stylish e-commerce website for clothing and accessories. It features categories for Men, Women, Accessories, and New Arrivals, with a fully interactive shopping experience including cart management, live search, and a responsive mobile layout.
This small project is a work-in-progress for experiments.

## Features

- **Responsive Navbar**: Easy navigation across categories with a hamburger menu for mobile.
- **Live Search**: Filter products across all categories in real time.
- **Shopping Cart**: Add products to cart with a live item count badge.
- **Product Listings**: Categorized sections for New Arrivals, Men, Women, and Accessories.
- **Smooth Scroll**: "Shop Now" on the hero banner scrolls directly to New Arrivals.
- **Newsletter Subscription**: Controlled form with email validation and a success confirmation.
- **Footer**: Quick links to all sections with dynamic copyright year.

## Tech Stack

- **Frontend**: React.js 18 with TypeScript
- **Styling**: Tailwind CSS
- **Package Manager**: npm
- **Build Tool**: Vite
